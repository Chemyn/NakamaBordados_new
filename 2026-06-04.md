## 🧠 Improvements & Learnings

📌 **New Rules / Discoveries**
* **GraphQL User Fetching**: El campo `customer` puede ser inestable vía JWT; usar una consulta combinada con `viewer` (WP) y `customer` (WC) al mismo nivel de raíz es más robusto para obtener perfiles completos sin errores 500.

🔄 **Optimizations & Reflections**
* **CSS Global vs Local**: Para componentes críticos de la UI (como botones flotantes con overlays complejos), es preferible usar estilos globales en `globals.css` para evitar problemas de prioridad y asegurar que las animaciones se rendericen uniformemente.
